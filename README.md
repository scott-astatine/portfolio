## ğŸ Getting Started

Ensure you have [Bun](https://bun.sh/) installed on your system.

### 1. Clone & Install
```bash
git clone https://github.com/scott-astatine/portfolio.git
cd portfolio
bun install
```

### 2. Development
```bash
bun dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 3. Build
```bash
bun run build
```
